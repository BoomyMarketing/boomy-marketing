# SunLeo — 27 Homepage Design Directions

Twenty-seven fully standalone homepage mockups for SunLeo, Calgary's custom blinds & shades brand, organised in four volumes: **01–10** concept previews · **11–17** TWOPAGES-depth e-commerce builds · **18–22** live & animated builds (GSAP/Lenis/CSS 3D) · **23–27** conversion-first mechanic-driven homepages. Each file is a complete, self-contained HTML page (TailwindCSS + Google Fonts via CDN, picsum.photos imagery). Open `INDEX.html` for the browsable gallery.

Each direction has its own conceptual through-line, distinctive typography pairing, custom palette, atmospheric background treatment, and orchestrated motion — built to feel like work from a top design studio rather than a generic template.

## What's different vs. a generic template

- **No Inter/Roboto/system defaults** — every design uses a distinctive type pairing.
- **No purple-gradient-on-white** — palettes are committed and one-of-a-kind (tomato/acid/cobalt, aurora multi-color, forest/copper/cream, sepia paper, etc.)
- **Custom backgrounds** — topographic SVG, blueprint grid, halftone dots, paper grain, checker, aurora blur, mountain silhouettes, repeating linen weave.
- **Custom UI primitives** — caliper corner ticks, ranger badges, washi tape, polaroid frames, plate captions, gradient borders, stamp postmarks, glass cards with gradient borders.
- **Orchestrated motion** — fade-up staggers (d1-d5), scan lines, marquees, throb pulses, draw-SVG paths, slow spins, drift floats.
- **Distinctive editorial voice** — luxury one uses "folio · plate · audience" language; architect uses "SL/01 · brief · field notes"; magazine uses "page 01 · letter · field notes · voices"; conversion one uses "60 seconds · slots left · countdown".

### What 23–27 intentionally do differently

- **Hero mechanic** is the *load-bearing conversion lever*, not just decoration — these aren't "homepages that happen to have a quiz/map/slider", they're "the quiz/map/slider IS the homepage"
- **No two designs share a typography pairing** — 23 (Old Standard TT + Spectral), 24 (Cormorant Garamond + Caveat), 25 (Fraunces + IBM Plex), 26 (Fraunces 900 + Archivo), 27 (Spectral + Caveat). Each pairing was chosen for the specific aesthetic of the mechanic.
- **No two designs share a colour palette** — 23 oxblood/cream/gold · 24 bone/olive/terracotta · 25 white/blue/yellow · 26 black/bone/red · 27 ivory/navy/red/sage
- **CRO copy tone** matches the mechanic — 23 uses editorial "diagnosis" language, 24 uses founder-first-person ("I do not sell windows"), 25 uses radical receipt-honest language with strike-throughs on competitors' hidden margins, 26 uses brutalist imperative voice ("DRAG. Don't IMAGINE."), 27 uses cartographic publication language ("Be the next red pin")
